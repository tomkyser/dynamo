## Decisions Made
- Registry uses internal EventEmitter for lifecycle events rather than requiring Switchboard at construction -- Switchboard integration deferred to wire.cjs factory (Plan 04) which composes all modules
- Write coordinator uses greedy batching: after dequeuing an item, it peeks at the next and merges if same table, continuing until a different table or empty queue
- Reconnection TTL implemented with setTimeout plus full cleanup in destroy() to prevent timer leaks in tests
- Write coordinator flush() is synchronous (no timers) -- enables deterministic testing and clean shutdown
