```python
class MCPClient:
    def __init__(self):
        self.url = os.environ.get("GRAPHITI_MCP_URL", MCP_URL)
        self.session_id = None

    def _initialize(self):
        if self.session_id: return
        resp = httpx.post(self.url, json={
            "jsonrpc": "2.0", "method": "initialize",
            "params": {"protocolVersion": "2025-03-26", "capabilities": {},
                       "clientInfo": {"name": "graphiti-helper", "version": "1.0"}},
            "id": 1
        }, headers={"Accept": "application/json, text/event-stream"}, timeout=5)
        self.session_id = resp.headers.get("mcp-session-id")
        # Send notifications/initialized
        headers = {"Content-Type": "application/json"}
        if self.session_id: headers["mcp-session-id"] = self.session_id
        httpx.post(self.url, json={"jsonrpc": "2.0", "method": "notifications/initialized"},
                   headers=headers, timeout=5)

    def call_tool(self, tool_name, arguments):
        self._initialize()
        headers = {"Content-Type": "application/json", "Accept": "application/json, text/event-stream"}
        if self.session_id: headers["mcp-session-id"] = self.session_id
        resp = httpx.post(self.url, json={
            "jsonrpc": "2.0", "method": "tools/call",
            "params": {"name": tool_name, "arguments": arguments},
            "id": str(uuid.uuid4())
        }, headers=headers, timeout=5)
        ct = resp.headers.get("content-type", "")
        if "text/event-stream" in ct:
            return self._parse_sse(resp.text)
        return resp.json()
```

**MCPClient class:**

```javascript
class MCPClient {
  constructor(options = {}) {
    const config = loadConfig();
    this.url = process.env.GRAPHITI_MCP_URL || options.url || config.graphiti?.mcp_url || MCP_DEFAULTS.url;
    this.timeout = options.timeout || config.timeouts?.mcp || MCP_DEFAULTS.timeout;
    this.sessionId = null;
  }
```

`async initialize()` method:
- If this.sessionId already set, return early (lazy init)
- Build headers: `{ 'Content-Type': 'application/json', 'Accept': 'application/json, text/event-stream' }`
- Step 1 -- Send initialize request:
  ```javascript
  const resp = await fetchWithTimeout(this.url, {
    method: 'POST',
    headers,
    body: JSON.stringify({
      jsonrpc: '2.0',
      method: 'initialize',
      params: {
        protocolVersion: '2025-03-26',
        capabilities: {},
        clientInfo: { name: 'dynamo', version: '0.1.0' }
      },
      id: 1
    })
  }, this.timeout);
  ```
- Extract session ID: `this.sessionId = resp.headers.get('mcp-session-id');`
- Step 2 -- Send notifications/initialized (MCP spec requires this):
  ```javascript
  const notifHeaders = { 'Content-Type': 'application/json' };
  if (this.sessionId) notifHeaders['mcp-session-id'] = this.sessionId;
  await fetchWithTimeout(this.url, {
    method: 'POST',
    headers: notifHeaders,
    body: JSON.stringify({
      jsonrpc: '2.0',
      method: 'notifications/initialized'
    })
  }, this.timeout);
  ```

`async callTool(toolName, args)` method:
- Call `await this.initialize()` first
- Build headers with Content-Type, Accept, and mcp-session-id (if available)
- Send JSON-RPC request:
  ```javascript
  const resp = await fetchWithTimeout(this.url, {
    method: 'POST',
    headers,
    body: JSON.stringify({
      jsonrpc: '2.0',
      method: 'tools/call',
      params: { name: toolName, arguments: args },
      id: crypto.randomUUID()
    })
  }, this.timeout);
  ```
- Check content-type header: if includes 'text/event-stream', parse with `parseSSE(await resp.text())`
- Otherwise return `await resp.json()`

**parseSSE(text) function (standalone, exported):**
```javascript
function parseSSE(text) {
  for (const line of text.split('\n')) {
    if (line.startsWith('data:')) {
      const data = line.slice(5).trim();
      if (data) {
        try {
          const parsed = JSON.parse(data);
          if ('result' in parsed || 'error' in parsed) return parsed;
        } catch { continue; }
      }
    }
  }
  return { error: { message: 'No valid JSON-RPC response in SSE stream' } };
}
```
