    `createMindCycle({ selfModel, formationPipeline, recallEngine, templateComposer, referentialFraming, sublimationLoop, switchboard, wire, lithograph, config } = {})`:

    This is the cognitive center per D-04 (full Mind minus REM). It orchestrates:
    - Attention management (deciding what deserves formation)
    - Formation orchestration (directing formation pipeline)
    - Recall (passive on every turn, explicit on keyword triggers)
    - Face prompt composition (including referential framing from D-12)
    - Sublimation evaluation (processing Tertiary candidates)
    - Directive generation (sending behavioral/attention directives to Primary)

    Internal state:
    - `_turnsProcessed = 0`
    - `_formationsInitiated = 0`
    - `_sublimationsEvaluated = 0`
    - `_lastTurnAt = null`
    - `_pendingSublimations = []` (queue of candidates waiting for evaluation)

    **processTurn({ userPrompt, toolUse, turnNumber })** (async):
    1. `_turnsProcessed++; _lastTurnAt = Date.now()`
    2. Run attention check: use formationPipeline.prepareStimulus({ userPrompt, toolUse, hookType: 'UserPromptSubmit' }) to get stimulus
       - If stimulus indicates below-threshold (empty or no domains), return `{ formed: false, reason: 'below_threshold' }`
    3. If attention-worthy: `_formationsInitiated++`, return `{ formed: true, stimulus, fragments_initiated: 1 }`
       - The actual formation subagent spawning happens externally (Session Manager triggers it)
    4. Run passive recall: `recallEngine.recallPassive({ userPrompt, turnNumber })` if recallEngine available
    5. Check for explicit recall keywords in userPrompt (RECALL_KEYWORDS regex: /\b(remember|recall|what did|last time|we discussed|you said)\b/i)
       - If match: also call `recallEngine.recallExplicit(userPrompt)`
    6. Generate directives object: `{ facePromptUpdate: composeFacePrompt(), recall: recallResult, behavioral: null }` (behavioral is null until Phase 11 conditioning)
    7. Return `ok({ formed, recall: recallResult, directives })`

    **processSublimation({ candidates, resonanceScores })** (async):
    1. `_sublimationsEvaluated += candidates.length`
    2. Filter candidates where resonanceScore >= sublimationLoop.getCycleConfig().sensitivity_threshold
    3. Cap at config.max_sublimation_intake (default 5) per D-08/Pitfall 4
    4. For worthy candidates, add to _pendingSublimations queue for next formation evaluation
    5. Return `ok({ evaluated: candidates.length, worthy: worthyCandidates.length, queued: _pendingSublimations.length })`

    **composeFacePrompt()** (sync):
    1. If templateComposer available, call templateComposer.compose with current budget phase
    2. The referential framing content should be available via referentialFraming.getPrompt()
    3. Return the composed face prompt string
    4. If templateComposer not available, return a minimal face prompt from Self Model identity

    **getState()**: Returns `{ turns_processed: _turnsProcessed, formations_initiated: _formationsInitiated, sublimations_evaluated: _sublimationsEvaluated, last_turn_at: _lastTurnAt, pending_sublimations: _pendingSublimations.length }`

    **drainSublimations()**: Returns and clears _pendingSublimations array (for formation pipeline to process)

    Export: `{ createMindCycle }`

    `createWireTopology({ wire, switchboard, config } = {})`:

    Internal state:
    - `_messagesSent = 0`
    - `_messagesBlocked = 0`
    - `_ackTimeouts = 0`
    - `_topologyViolations = 0`
    - `_pendingAcks = new Map()` -- envelopeId -> { resolve, timer }

    Constants for ACK-required message types:
    ```javascript
    const ACK_REQUIRED_TYPES = new Set([
      MESSAGE_TYPES.CONTEXT_INJECTION,
      MESSAGE_TYPES.DIRECTIVE,
    ]);
    const ACK_REQUIRED_URGENCIES = new Set([
      URGENCY_LEVELS.URGENT,
      URGENCY_LEVELS.DIRECTIVE,
    ]);
    ```

    **validateRoute(fromIdentity, toIdentity)**:
    - Look up TOPOLOGY_RULES[fromIdentity]
    - If toIdentity not in allowed list, return `err('TOPOLOGY_VIOLATION', \`${fromIdentity} cannot send to ${toIdentity}\`, { allowed: TOPOLOGY_RULES[fromIdentity] })`
    - `_topologyViolations++` on violation
    - Return `ok({ from: fromIdentity, to: toIdentity })`

    **send(envelope)** (async):
    - Extract from/to identities from envelope (look up in Wire registry or use the identity field directly)
    - Call validateRoute. If err, `_messagesBlocked++`, return the error
    - Determine if ACK is required: `ACK_REQUIRED_TYPES.has(envelope.type) && ACK_REQUIRED_URGENCIES.has(envelope.urgency)`
    - If ACK required: delegate to `sendWithAck(envelope)`
    - Otherwise: call `wire.send(envelope)`, `_messagesSent++`, return result

    **sendWithAck(envelope)** (async):
    - Call `wire.send(envelope)`, `_messagesSent++`
    - Wait for ACK: `waitForAck(envelope.id, config.ack_timeout_ms || 5000)`
    - If ACK received: return ok
    - If timeout: `_ackTimeouts++`, attempt resend via wire.send with `_forceRelay: true` flag on envelope
    - Return result (ok if resend succeeds, err if both attempts fail)

    **waitForAck(envelopeId, timeoutMs)** (returns Promise):
    - Create a Promise
    - Set timer for timeoutMs
    - Store { resolve, timer } in _pendingAcks keyed by envelopeId
    - When ACK message with correlationId === envelopeId arrives (via _handleIncomingAck), resolve the Promise
    - On timeout: reject with err('ACK_TIMEOUT')

    **_handleIncomingAck(envelope)** (internal):
    - If envelope.type === MESSAGE_TYPES.ACK and _pendingAcks.has(envelope.correlationId):
    - clearTimeout(timer), resolve(ok()), delete from _pendingAcks

    **subscribe(sessionId, callback)** (wraps wire.subscribe):
    - Determine subscriber's identity from sessionId (look up in context or accept as param)
    - Call `wire.subscribe(sessionId, (envelope) => { ... })`
    - In the wrapper callback:
      - Check if envelope.type === 'ack': route to _handleIncomingAck
      - Validate sender is in TOPOLOGY_RULES for this subscriber's identity
      - If valid: call callback(envelope)
      - If not: log/emit topology violation, do not deliver
    - Return unsubscribe function

    **getMetrics()**: Returns `{ messages_sent: _messagesSent, messages_blocked: _messagesBlocked, ack_timeouts: _ackTimeouts, topology_violations: _topologyViolations }`

    Export: `{ createWireTopology }`
