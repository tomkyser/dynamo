<objective>
Build the Circuit module API -- the SDK layer that modules (like Reverie) consume to access Dynamo platform services safely.

Purpose: Circuit is the only gateway for modules to interact with the platform. Modules never import from lib/ directly or access the container. Everything a module needs comes through Circuit: facade-only service/provider access (D-01), lib essentials re-export (D-02), manifest-based registration (D-03), and namespaced event proxy (D-04).

Output: Three source files (event-proxy.cjs, module-manifest.cjs, circuit.cjs) with full test coverage.
</objective>

  <action>
    Create `core/sdk/circuit/circuit.cjs`:
    - `'use strict'` header
    - Require lib essentials: `const { ok, err, isOk, isErr, createContract, validate } = require('../../../lib/index.cjs');`
    - Require local modules: `const { createEventProxy } = require('./event-proxy.cjs');`
    - Require manifest: `const { validateModuleManifest } = require('./module-manifest.cjs');`

    - Define `CIRCUIT_SHAPE`:
      ```
      { required: ['registerModule', 'shutdownModule', 'getService', 'getProvider', 'getModuleInfo', 'listModules'],
        optional: ['registerCommand', 'registerMcpTool'] }
      ```

    - `createCircuit(options = {})` function:
      - Extract `lifecycle`, `container`, `pulley` from options
      - `const _modules = new Map()` -- stores `{ manifest, eventProxy }` per module name

      - `registerModule(manifest, registerFn)`:
        1. Validate manifest via `validateModuleManifest(manifest)`. Return err if invalid.
        2. Check `_modules.has(manifest.name)` -- return `err('MODULE_EXISTS', ...)` if duplicate.
        3. Check dependencies: iterate `manifest.dependencies.services`, verify `container.has('services.' + svc)` for each. Iterate `manifest.dependencies.providers`, verify `container.has('providers.' + prov)` for each. Collect missing. Return `err('MODULE_MISSING_DEPS', ...)` if any missing.
        4. Create event proxy: `const eventProxy = createEventProxy(manifest.name, lifecycle.getFacade('services.switchboard'))`.
        5. Build scoped circuit API object passed to registerFn:
           ```
           {
             getService: (name) => _scopedGetService(manifest.name, name),
             getProvider: (name) => _scopedGetProvider(manifest.name, name),
             events: eventProxy,
             registerCommand: pulley ? (name, handler, meta) => pulley.registerCommand(`${manifest.name} ${name}`, handler, meta) : () => err('NO_PULLEY', 'Pulley not available'),
             registerMcpTool: pulley ? (name, handler, schema) => pulley.registerMcpTool(name, handler, schema) : () => err('NO_PULLEY', 'Pulley not available'),
             ok, err, isOk, isErr, validate, createContract,
           }
           ```
        6. Call `registerFn(circuitApi)`. If it returns an Err result, propagate.
        7. Store `_modules.set(manifest.name, { manifest, eventProxy })`.
        8. Return `ok({ name: manifest.name })`.

      - `_scopedGetService(moduleName, serviceName)`:
        - Look up module in `_modules`. Return err if not registered.
        - Check `module.manifest.dependencies.services.includes(serviceName)`. Return `err('UNDECLARED_DEPENDENCY', ...)` if not declared.
        - Return `lifecycle.getFacade('services.' + serviceName)` wrapped in ok/err.

      - `_scopedGetProvider(moduleName, providerName)`:
        - Same pattern as _scopedGetService but with providers.

      - `getService(moduleName, serviceName)`: calls `_scopedGetService`.
      - `getProvider(moduleName, providerName)`: calls `_scopedGetProvider`.

      - `shutdownModule(moduleName)`:
        - Look up module. Call `eventProxy.cleanup()`. Remove from `_modules`. Return ok.

      - `getModuleInfo(moduleName)`:
        - Look up module. Return `ok({ name, manifest, subscriptions: eventProxy.getSubscriptionCount() })`.

      - `listModules()`:
        - Return `ok([..._modules.keys()])`.

      - `registerCommand(...)` and `registerMcpTool(...)`:
        - Delegate to pulley if available, err otherwise.

      - Return via `createContract('circuit', CIRCUIT_SHAPE, { registerModule, shutdownModule, getService, getProvider, getModuleInfo, listModules, registerCommand, registerMcpTool })`.

    - `module.exports = { createCircuit, CIRCUIT_SHAPE }`.

    Per D-01: facade-only access via lifecycle.getFacade. Per D-02: lib essentials re-exported. Per D-03: manifest + register callback. Per D-04: event proxy with namespacing.
