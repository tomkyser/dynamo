## Decisions Made
- Used node:crypto.randomUUID() for envelope ID generation -- zero-dependency, Bun-native
- Queue uses simple array shift/push for FIFO within urgency buckets -- O(1) amortized, no external data structure needed
- Urgent queue has Infinity depth limit per D-09 -- urgent messages are never dropped
- Default depth limits: background=50, active=200, directive=100 -- all configurable via constructor options
