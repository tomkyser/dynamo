    **session-spawner.cjs** (core/services/conductor/session-spawner.cjs):

    Create with `'use strict'` header. Factory function `createSessionSpawner(options = {})`.

    Options: `{ channelServerPath, switchboard }` -- channelServerPath is the path to Wire's channel-server.cjs.

    Internal `_sessions = new Map()` tracks spawned sessions by sessionId.

    Methods:

    1. `spawn({ sessionId, identity, env = {} })`:
       - Construct args: `['claude', '--dangerously-load-development-channels', 'server:' + channelServerPath]`
       - Merge env: `{ ...process.env, WIRE_RELAY_URL: env.relayUrl || '', SESSION_ID: sessionId, SESSION_IDENTITY: identity, ...env }`
       - Call `Bun.spawn(args, { env: mergedEnv, stdin: 'pipe', stdout: 'pipe', stderr: 'pipe' })`
       - Store in _sessions: `_sessions.set(sessionId, { proc, identity, startedAt: Date.now() })`
       - Emit via switchboard if available: `switchboard.emit('infra:session-spawned', { sessionId, identity, pid: proc.pid })`
       - Return `ok({ sessionId, pid: proc.pid, proc })`

    2. `stop(sessionId)`:
       - Lookup in _sessions. If not found, return `err('SESSION_NOT_FOUND', ...)`
       - Call `proc.kill()` (SIGTERM)
       - Remove from _sessions
       - Emit `'infra:session-stopped'`
       - Return `ok({ sessionId })`

    3. `health(sessionId)`:
       - Lookup in _sessions. If not found, return `err('SESSION_NOT_FOUND', ...)`
       - Check `!proc.killed` and `proc.exitCode === null` for alive status
       - Return `ok({ sessionId, alive, pid: proc.pid, uptime: Date.now() - startedAt })`

    4. `list()` -- returns array of { sessionId, identity, alive } for all tracked sessions

    Export: `{ createSessionSpawner }`
