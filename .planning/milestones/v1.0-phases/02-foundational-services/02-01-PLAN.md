<objective>
Build the Lathe filesystem facade service -- the lowest-level service in the dependency chain with zero service dependencies. Lathe wraps Bun.file, Bun.write, and node:fs into a single facade that all other services and future code use for filesystem access.

Purpose: Lathe is the filesystem primitive. Magnet's JSON provider (Plan 03) depends on it for persistence. Forge, Relay, and Journal (Phase 3+) depend on it for all file I/O. Building it first unblocks everything.
Output: `core/services/lathe/lathe.cjs` with full test coverage via `core/services/lathe/__tests__/lathe.test.js`
</objective>
