      const WIRE_SHAPE = {
        required: [
          'init', 'start', 'stop', 'healthCheck',

    - `createWire()` factory function:
      - Private state: `_started = false`, `_switchboard = null`, `_conductor = null`, `_ledger = null`, `_registry = null`, `_queue = null`, `_writeCoordinator = null`, `_transportRouter = null`, `_subscribers = new Map()` (Map<sessionId, callback[]>)
      - `init(options)`:
        - Extract `{ switchboard, conductor, ledger, relayUrl, reconnectTTL, queueConfig }` from options
        - Store `_switchboard`, `_conductor`, `_ledger`
        - Create `_registry = createRegistry({ reconnectTTL: reconnectTTL || 30000 })`
        - Create `_queue = createPriorityQueue(queueConfig || {})`
        - Create `_writeCoordinator = createWriteCoordinator({ ledger: _ledger, queueConfig })`
        - Create transports: `_channelsTransport = createChannelsTransport({ mcpServer: options.mcpServer || null })`, `_relayTransport = createRelayTransport({ relayUrl: relayUrl || 'http://127.0.0.1:9876', sessionId: options.sessionId, onMessage: _handleIncomingMessage })`
        - Create `_transportRouter = createTransportRouter([_channelsTransport, _relayTransport])`
        - Wire registry lifecycle events to Switchboard: `_registry.on('session:registered', (data) => _switchboard && _switchboard.emit('wire:session-registered', data))`, same for `session:lost` -> `wire:session-lost`, `session:reconnected` -> `wire:session-reconnected`
        - Wire write coordinator events: `_writeCoordinator.on('write:completed', (data) => _switchboard && _switchboard.emit('wire:write-completed', data))`, same for `write:failed` -> `wire:write-failed`
        - Return `ok(undefined)`
      - `start()`:
        - Connect transports via `_transportRouter.connectAll()`
        - Start write coordinator via `_writeCoordinator.start()`
        - Set `_started = true`
        - `_switchboard && _switchboard.emit('wire:started', {})`
        - Return `ok(undefined)`
      - `stop()`:
        - Stop write coordinator
        - Disconnect transports
        - Destroy registry (clear timers)
        - Set `_started = false`
        - `_switchboard && _switchboard.emit('wire:stopped', {})`
        - Return `ok(undefined)`
      - `healthCheck()`:
        - Return `ok({ started: _started, sessions: _registry ? _registry.getSessions().length : 0, writeQueueDepth: _writeCoordinator ? _writeCoordinator.getQueueDepth().total : 0, transports: _transportRouter ? _transportRouter.healthCheck() : [] })`
      - `send(envelope)`:
        - Validate envelope, route through `_transportRouter.send(envelope)`
        - `_switchboard && _switchboard.emit('wire:message-sent', { from: envelope.from, to: envelope.to, type: envelope.type, urgency: envelope.urgency })`
        - Return Result
      - `subscribe(sessionId, callback)`:
        - Add callback to `_subscribers` Map for sessionId
        - Return unsubscribe function
      - `register(sessionId, info)`:
        - Delegate to `_registry.register(sessionId, info)`
        - Return `ok(undefined)`
      - `unregister(sessionId)`:
        - Delegate to `_registry.unregister(sessionId)`
        - Remove subscribers for sessionId
        - Return `ok(undefined)`
      - `getRegistry()`: return `_registry`
      - `queueWrite(envelope)`:
        - Delegate to `_writeCoordinator.queueWrite(envelope)`
        - `_switchboard && _switchboard.emit('wire:write-queued', { from: envelope.from })`
        - Return Result
      - `broadcast(envelope)`: send to all registered sessions
      - `getQueueDepth()`: delegate to `_writeCoordinator.getQueueDepth()`
      - `flush()`: delegate to `_writeCoordinator.flush()`
      - `createEnvelope(params)`: re-export from protocol for convenience
      - `_handleIncomingMessage(envelope)`: look up subscribers for `envelope.to`, call each with envelope. If session is disconnected, buffer via registry.
    - `return createContract('wire', WIRE_SHAPE, impl)`

      - Register `ListToolsRequestSchema` handler with 3 tools:
        1. `wire_send`: params `{ to (string), type (string enum of MESSAGE_TYPES values), urgency (string enum of URGENCY_LEVELS values), payload (string), correlationId (string, optional) }`. Description: "Send a message to another session via Wire."
        2. `wire_register`: params `{ identity (string), capabilities (string[]), writePermissions (string[]) }`. Description: "Register this session with Wire for message routing."
        3. `wire_status`: no params. Description: "Get Wire connection status and registered sessions."
      - Register `CallToolRequestSchema` handler:
        - `wire_send`: create envelope with `createEnvelope({ from: _sessionId, to, type, urgency, payload: JSON.parse(payload) })`. POST to relay `/send` endpoint. Push channel notification via `mcp.notification({ method: 'notifications/claude/channel', params: { content: 'Message sent to ' + to, meta: { status: 'sent', to_session: to, message_type: type } } })`. Return tool result with delivery status.
        - `wire_register`: POST to relay `/register` with `{ sessionId: _sessionId, identity, capabilities, writePermissions }`. Return confirmation.
        - `wire_status`: GET relay `/health`. Return formatted status.
      - `_startPolling()`: set interval to GET relay `/poll?sessionId=${_sessionId}&timeout=5000`. On messages, for each message call `_onMessage(message)` and push channel notification: `mcp.notification({ method: 'notifications/claude/channel', params: { content: JSON.stringify(message.payload), meta: { from_session: message.from, urgency_level: message.urgency, message_type: message.type, message_id: message.id } } })`.
      - `_stopPolling()`: clear interval.
      - `start()`: connect MCP via `StdioServerTransport`, start polling. (async IIFE wrapper for safety)
      - `stop()`: stop polling, close MCP.
