    1. Create `modules/reverie/components/rem/quality-evaluator.cjs`:

    Core functions:
    - `computeBehavioralScore(sessionMetrics)` — takes `{ turn_count, avg_turn_length, session_duration_ms, recall_events, recall_incorporated, directive_compliance_rate, friction_signals }`. Computes weighted composite:
      - turn_engagement: min(turn_count / 30, 1.0) * 0.2
      - depth: min(avg_turn_length / 200, 1.0) * 0.2
      - duration: min(session_duration_ms / 1800000, 1.0) * 0.15 (30min baseline)
      - recall_usage: (recall_incorporated / max(recall_events, 1)) * 0.15
      - compliance: directive_compliance_rate * 0.15
      - friction_inverse: (1 - min(friction_signals / 5, 1.0)) * 0.15
      Returns clamped 0.0-1.0.

    - `composeLlmReflectionPrompt(sessionSummary, conditioningState)` — returns a string prompt asking the Mind (Secondary LLM) to rate session quality 0.0-1.0 with justification. The prompt includes session summary, current conditioning state, and asks about: personality-mood fit, conversation flow, information retrieval effectiveness, user satisfaction signals.

    - `parseLlmScore(llmResponse)` — extracts a numeric 0.0-1.0 score from LLM text response. Looks for patterns like "score: 0.7" or "0.7/1.0" or just a decimal. Returns null if unparseable.

    Factory `createQualityEvaluator({ entropyEngine, config } = {})`:

    `evaluateSession(sessionMetrics, llmScore)`:
