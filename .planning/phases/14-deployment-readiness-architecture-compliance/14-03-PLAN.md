**Criterion 2: No hardcoded paths/values that should route through config/Magnet/providers**

Search for violations:
- grep for hardcoded file paths like `/data/ledger.db` or `/data/journal` in module code -- should come from config or paths
- grep for hardcoded `.claude/` paths in module code -- should come from Exciter
- grep for magic numbers that should be in constants (e.g., token budgets, retry counts, timeout values not in a CONSTANTS object)
- NOTE: Ignore test files -- mocks and fixtures legitimately hardcode values
- NOTE: Ignore lib/constants.cjs files -- these ARE the config source

- Services only do (no data storage in service code)
- Providers only supply and receive (no business logic)
- No direct LLM API calls below SDK scope
