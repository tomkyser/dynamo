    1. computeCapPressure(domainCount, maxEntityCount, edgeCount):
       ```javascript
       function computeCapPressure(domainCount, maxEntityCount, edgeCount) {
         const caps = _config.taxonomy || TAXONOMY_DEFAULTS;
         return {
           domainCount,
           maxEntityCount,
           edgeCount,
           domainPressure: domainCount / caps.max_domains,
           entityPressure: maxEntityCount / caps.max_entities_per_domain,
           edgePressure: edgeCount / caps.max_association_edges,
           isUnderPressure: (domainCount / caps.max_domains) >= caps.pressure_threshold
             || (maxEntityCount / caps.max_entities_per_domain) >= caps.pressure_threshold
             || (edgeCount / caps.max_association_edges) >= caps.pressure_threshold,
         };
       }
       ```

    2. getPressureGradientText(capPressure):
       - Returns urgency text for editorial prompt based on pressure level

       - At 95%+: "URGENT: Near taxonomy cap. Merge near-synonyms with minimal justification. Retire all inactive domains."
       - Below 80%: "" (no pressure text)
