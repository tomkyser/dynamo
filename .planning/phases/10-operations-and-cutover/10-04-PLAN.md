    **Part B: install.cjs installer**

    Create `dynamo/lib/switchboard/install.cjs` with module identity block `// Dynamo > Switchboard > install.cjs`.

    Require: `path`, `fs`, `os`, `{ execSync }` from `child_process`. Require core.cjs for `{ DYNAMO_DIR, output, error, loadEnv, safeReadFile }`. Require pretty.cjs for `{ formatInstallReport }`.

    Define constants:
    ```javascript
    const REPO_DIR = path.join(__dirname, '..', '..');         // dynamo/ in repo
    const LIVE_DIR = path.join(os.homedir(), '.claude', 'dynamo');
    const GRAPHITI_DIR = path.join(os.homedir(), '.claude', 'graphiti');
    const LEGACY_DIR = path.join(os.homedir(), '.claude', 'graphiti-legacy');
    const SETTINGS_PATH = path.join(os.homedir(), '.claude', 'settings.json');
    const SETTINGS_BACKUP = SETTINGS_PATH + '.bak';
    const HOOKS_TEMPLATE = path.join(REPO_DIR, '..', 'claude-config', 'settings-hooks.json');
    const INSTALL_EXCLUDES = ['tests', '.last-sync', '.git', '.DS_Store', 'node_modules'];
    ```

    Implement helper functions:

    **copyTree(src, dst, excludes):** Recursively copy directory tree, skipping excluded names. Create directories with `fs.mkdirSync(dst, { recursive: true })`. For each file: `fs.copyFileSync(srcPath, dstPath)`.

    **generateConfig(envPath):** Read .env via loadEnv. Build config.json object:
    ```javascript
    {
      version: safeReadFile(path.join(REPO_DIR, 'VERSION')).trim(),
      graphiti: {
        mcp_url: env.GRAPHITI_MCP_URL || 'http://localhost:8100/mcp',
        health_url: env.GRAPHITI_HEALTH_URL || 'http://localhost:8100/health'
      },
      curation: {
        model: 'anthropic/claude-haiku-4.5',
        api_url: env.OPENROUTER_API_URL || 'https://openrouter.ai/api/v1/chat/completions'
      },
      timeouts: { health: 3000, mcp: 5000, curation: 10000, summarization: 15000 },
      logging: { max_size_bytes: 1048576, file: 'hook-errors.log' }
    }
    ```
    Write to `path.join(LIVE_DIR, 'config.json')`.

    **mergeSettings(settingsPath, templatePath):**
    1. Read existing settings.json (or start with `{}`).
    2. **BACKUP FIRST:** `fs.copyFileSync(settingsPath, SETTINGS_BACKUP)`. If settings.json doesn't exist yet, skip backup.
    3. Read hooks template from templatePath.
    4. Replace `settings.hooks` with template hooks (template is authoritative for Dynamo hooks). BUT: preserve any non-Dynamo hooks (hooks whose command does NOT contain `dynamo-hooks.cjs`). Strategy: for each event in existing hooks, keep entries whose command doesn't contain `dynamo-hooks.cjs`, then add all template entries.
    5. Merge permissions additively: add template `permissions.allow` items not already present. Same for `permissions.ask`.
    6. Merge env: `Object.assign(settings.env || {}, template.env)`.
    7. Write atomically: `fs.writeFileSync(settingsPath + '.tmp', JSON.stringify(settings, null, 2) + '\n')`, then `fs.renameSync(settingsPath + '.tmp', settingsPath)`.

    **registerMcp():** Try `execSync('claude mcp add --transport http --scope user graphiti http://localhost:8100/mcp', { timeout: 10000, stdio: ['pipe','pipe','pipe'] })`. Catch errors and check if already registered (read `~/.claude.json`, look for `mcpServers.graphiti`).

    **retirePython():** Move Python-specific files from graphiti/ to graphiti-legacy/:
    - Files to move: `*.py`, `.venv/`, `__pycache__/`, `requirements.txt`, `curation/`, `hooks/*.sh` (shell hooks)
    - Files to KEEP in graphiti/: `docker-compose.yml`, `.env`, `.env.example`, `config.yaml`, `sessions.json`, `hook-errors.log`
    - Create graphiti-legacy/ if needed. Use `fs.renameSync` for moves.

    Implement `async function run(args = [], pretty = false)`:
    1. Parse args for `--pretty`.
    2. Define steps array for reporting:
    3. **Step 1 - Copy files:** `copyTree(REPO_DIR, LIVE_DIR, INSTALL_EXCLUDES)`. Record file count.
    4. **Step 2 - Generate config:** `generateConfig(path.join(GRAPHITI_DIR, '.env'))`.
    5. **Step 3 - Merge settings:** `mergeSettings(SETTINGS_PATH, HOOKS_TEMPLATE)`.
    6. **Step 4 - Register MCP:** `registerMcp()`.
    7. **Step 5 - Retire Python:** `retirePython()`.
    8. **Step 6 - Post-install health check:** `const hc = require('./health-check.cjs'); await hc.run([], false)`. Capture result but don't fail install on health check failure.
    9. Build result: `{ command: 'install', steps: [...], summary: { completed, total }, health_check: hcResult }`.
    10. If pretty: `formatInstallReport(result)`. Always: `output(result)`.

    Implement `async function rollback(args = [], pretty = false)`:
    1. **Restore settings.json:** If `SETTINGS_BACKUP` exists, `fs.copyFileSync(SETTINGS_BACKUP, SETTINGS_PATH)`.
    2. **Restore Python files:** If `LEGACY_DIR` exists, move files from graphiti-legacy/ back to graphiti/. Then `fs.rmdirSync(LEGACY_DIR)` or leave if not empty.
    3. Output: `{ command: 'rollback', status: 'ok', restored: ['settings.json', 'graphiti/'] }`.

    Export: `module.exports = { run, rollback }`.
