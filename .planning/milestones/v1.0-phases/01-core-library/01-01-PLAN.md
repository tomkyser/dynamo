    Import: `const { ok, err } = require('./result.cjs');`

    Schema field typedef (JSDoc):
    ```
    @typedef {Object} SchemaField
    @property {'string'|'number'|'boolean'|'object'|'array'} type
    @property {boolean} [required] - defaults to false
    @property {*} [default] - default value when field is missing
    @property {Object<string, SchemaField>} [properties] - for nested object validation
    ```

    Export `validate(value, schema)`:
    - If `value` is null or not an object, return `err('SCHEMA_INVALID_ROOT', 'Value must be a non-null object')`.
    - Iterate `Object.entries(schema)`. For each `[key, field]`:
      - If `value[key]` is undefined: check `field.required` (error) or `field.default` (apply default) or skip.
      - Type check: for `field.type === 'array'` use `Array.isArray()`. For others, use `typeof value[key] !== field.type`.
      - Nested: if `field.type === 'object'` and `field.properties` exists, recurse. Prefix nested error keys with `${key}.`.
      - Accumulate ALL errors (do not return on first error).
    - If errors array is non-empty, return `err('SCHEMA_VALIDATION_FAILED', 'Validation failed', { errors })`.
    - Otherwise return `ok(result)` with only validated/defaulted keys (strip extra keys).

    Module pattern:
    ```javascript
    'use strict';
    const { ok, err } = require('./result.cjs');
    // JSDoc typedefs
    function validate(value, schema) { ... }
    module.exports = { validate };
    ```
