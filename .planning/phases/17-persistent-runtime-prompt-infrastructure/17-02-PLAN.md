Create lib/linotype/composer.cjs with two exported functions:

1. `compose(slugs, options)`:
   - `slugs`: Array of Slug objects (from cast()).
   - `options`: `{ separator: string (default '\n\n'), token_budget: number|null (default null) }`.
   - Join slug contents with separator.
   - Calculate total_tokens as sum of all slug token_estimates.
   - Build sections array: `[{ name: slug.name, tokens: slug.token_estimate }]` for each slug.
   - If token_budget provided, compute budget_remaining = token_budget - total_tokens.
   - Return `createForme({ content, total_tokens, sections, budget: token_budget, budget_remaining })`.
   - If token_budget provided and total_tokens exceeds it, log a warning to stderr but do NOT throw (budget is advisory per PRD).

2. `inspect(forme)`:
   - Takes a Forme object.
   - Returns a plain object (NOT frozen -- it's debug output): `{ sections: forme.sections, total_tokens: forme.total_tokens, budget: forme.budget, budget_remaining: forme.budget_remaining }`.
   - This is the bill of materials per D-15.

Create lib/linotype/validator.cjs with one exported function:

1. `validate(matrix, options)`:
   - `matrix`: A Matrix object.
   - `options`: `{ registry: Map|null }` -- registry of known template names for partial validation.
   - Returns an array of issue objects: `[{ field: string, issue: string }]`. Empty array = no issues.
   - Checks:
     a. `name` is non-empty string. Issue: `{ field: 'name', issue: 'missing or empty' }`.
     b. `version` is non-empty string. Issue: `{ field: 'version', issue: 'missing or empty' }`.
     c. Every slot in `slots` has a `required` boolean field. Issue: `{ field: 'slots.{slotName}', issue: 'missing "required" field' }`.
     d. Every slot with `type` has a valid type string ('string', 'array', 'number', 'boolean'). Issue: `{ field: 'slots.{slotName}.type', issue: 'invalid type "{value}"' }`.
     e. If `includes` array is non-empty and registry provided, every include name exists in registry. Issue: `{ field: 'includes', issue: 'unresolved partial "{name}"' }`.
     f. Body syntax well-formedness: every `{{#if` has matching `{{/if}}`, every `{{#each` has matching `{{/each}}`. Issue: `{ field: 'body', issue: 'unmatched {{#if}} block' }`.

Module pattern: `'use strict'` at top, `module.exports = { compose, inspect }` and `module.exports = { validate }` respectively.

Exported functions:

1. `parse(filePath)` -- delegates to parser.parse(filePath). Returns Promise<Matrix>.

2. `parseString(content, sourceName)` -- delegates to parser.parseString(). Returns Matrix.

3. `cast(matrix, context, options)`:
   - `matrix`: Matrix object (from parse/parseString).
   - `context`: Object of slot values.
   - `options`: `{ partials: Map }` (optional, for include resolution).
   - Calls `resolve(matrix.body, context, { slots: matrix.slots, partials: options?.partials || new Map(), strict: true })`.
   - Computes `resolved_slots`: array of slot names from matrix.slots that have values in context.
   - Returns `createSlug({ name: matrix.name, content: resolvedBody, resolved_slots, token_estimate: Math.ceil(resolvedBody.length / 4) })`.

4. `compose(slugs, options)` -- delegates to composer.compose().

5. `validate(matrix, options)` -- delegates to validator.validate().

6. `inspect(forme)` -- delegates to composer.inspect().

Module exports: `module.exports = { parse, parseString, cast, compose, validate, inspect }`.
