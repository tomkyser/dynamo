    **Internal helper `_runGit(args, cwd)`:**
    ```javascript
    function _runGit(args, cwd) {
      const result = Bun.spawnSync(['git', ...args], {
        cwd: cwd || _repoPath,
        env: { ...process.env, GIT_TERMINAL_PROMPT: '0' },
        stdin: 'ignore',
      });
      if (!result.success) {
        const stderr = result.stderr.toString().trim();
        return err('GIT_FAILED', `git ${args[0]} failed: ${stderr}`, {
          command: ['git', ...args],
          exitCode: result.exitCode,
        });
      }
      return ok(result.stdout.toString().trim());
    }
    ```

    **Lifecycle methods:**
    - `init(options)`: Requires `options.repoPath`. Store `_lathe`, `_switchboard`. Check git is available: `_runGit(['--version'])`. If git not available, return err('INIT_FAILED', 'git is not installed'). Return ok(undefined).
    - `start()`: `_started = true`, return ok(undefined)
    - `stop()`: `_started = false`, return ok(undefined)
    - `healthCheck()`: Check git available via `_runGit(['--version'])`. Return ok({ healthy: _started, name: 'forge', gitAvailable: gitResult.ok })

    **Git operations:**
    - `status()`: Run `_runGit(['status', '--porcelain'])`. Parse output: empty = clean. Non-empty = split by newline, map each line to `{ status: line.slice(0, 2).trim(), file: line.slice(3) }`. Return ok({ clean: files.length === 0, files }).
    - `stageAll()`: Run `_runGit(['add', '-A'])`. Return result.
    - `stageFiles(files)`: Run `_runGit(['add', '--', ...files])`. Return result.
    - `commit(message)`: Run `_runGit(['commit', '-m', message])`. Parse hash from output (look for commit hash pattern). If output contains 'nothing to commit', return err('NOTHING_TO_COMMIT', ...). Emit 'git:committed' via switchboard: `{ hash, message }`. Return ok({ hash }).
    - `branch(name?)`: If no name, run `_runGit(['branch', '--show-current'])` and return ok(output). If name provided, run `_runGit(['checkout', '-b', name])` and return ok(name).
    - `tag(name)`: Run `_runGit(['tag', name])`. If fails with 'already exists', return err('TAG_EXISTS', ...). Emit 'git:tagged': `{ tag: name }`. Return ok(name).
    - `deleteTag(name)`: Run `_runGit(['tag', '-d', name])`. Return result.
    - `log(limit = 10)`: Run `_runGit(['log', '--format=%H|%s|%aI', '-n', String(limit)])`. Parse output: split by newline, split each line by '|' into { hash, message, date }. Return ok(entries).
    - `resetTo(ref)`: Run `_runGit(['reset', '--hard', ref])`. Return result.

    **Submodule operations:**
    - `submoduleAdd(url, subPath)`: Run `_runGit(['submodule', 'add', url, subPath])`. Return result.
    - `submoduleUpdate()`: Run `_runGit(['submodule', 'update', '--init', '--recursive'])`. Return result.
    - `submoduleRemove(subPath)`: Run sequence: (1) `_runGit(['submodule', 'deinit', '-f', subPath])`, (2) `_runGit(['rm', '-f', subPath])`. If .gitmodules still references it, clean up. Return result of last command.

    **Sync operations:**
    - `async sync(srcDir, destDir)`: List files in srcDir via `_lathe.listDir(srcDir)`. For each file, read from src, write to dest via `_lathe.readFile` and `_lathe.writeFile`. For directories, recurse. Return ok({ filesCopied: count }).

    Return `createContract('forge', FORGE_SHAPE, impl)` where impl includes all methods.
