## Decisions Made
- Used `Object.freeze` on PARSERS registry for runtime immutability (matches platform convention from Phase 7)
- Included `setTranscriptPath` in the impl object passed to `validateDataProvider` — `createContract` spreads and freezes all properties, so the extra method is preserved alongside the standard DATA_PROVIDER_SHAPE methods
- Sync `fs.readFileSync` for read/query operations (D-04: hooks need fast response) — async only for write/delete via Lathe
- Strip `_parserVersion` and `_lineIndex` metadata before serializing back to JSONL to avoid polluting the transcript file
