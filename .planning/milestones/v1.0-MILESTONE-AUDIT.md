### Integration Gaps (3 issues, all in core/core.cjs bootstrap wiring)

**INT-GAP-01 (High): Assay provider injection**
- `services.assay` has `deps: []` in core/core.cjs — boots in level-0 before Ledger/Journal
- Receives uninitialized provider instances via container.resolve() fallback
- `assay.search()` silently returns empty results (errors caught by Promise.allSettled)
- **Fix:** Add `'providers.ledger', 'providers.journal'` to `services.assay` deps[]
- Affects: SVC-09, PRV-01, PRV-02

**INT-GAP-02 (Medium): Magnet state persistence not wired**
- `createJsonProvider` built and tested in Phase 2 but never imported in core/core.cjs
- Magnet receives `provider = undefined` — all state is in-memory only
- **Fix:** Wire createJsonProvider into Magnet's bootstrap configuration
- Affects: SVC-03

**INT-GAP-03 (Medium): forge.pull() missing**
- `handleUpdate` in platform-commands.cjs calls `forge.pull('origin', 'master')`
- Method not implemented in forge.cjs; typeof guard silently skips the step
- `dynamo update` never pulls core from remote
- **Fix:** Implement `pull(remote, branch)` in forge.cjs
- Affects: SVC-05, INF-02
