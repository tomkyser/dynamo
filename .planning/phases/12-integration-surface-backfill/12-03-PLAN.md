       Per D-04, create 3 handler functions, ALL with --confirm gate:

       Helper function for confirm check (used by all 3 handlers):
       ```javascript
       function _requireConfirm(flags) {
         // Check for --confirm in flags (Pulley passes parsed flags)
         // Since Pulley only parses --json/--raw/--help, check process.argv directly
         const hasConfirm = process.argv.includes('--confirm');
         if (!hasConfirm) {
           return err('CONFIRM_REQUIRED', 'This operation is destructive. Add --confirm to proceed.', {
             hint: 'dynamo reverie reset <scope> --confirm',
           });
         }
         return null; // confirmed
       }
       ```

       a. handleResetFragments(args, flags):
          - Check _requireConfirm(flags). If err, return it immediately.
          - List all fragments across working/ active/ archive/ directories via journal
          - Delete each fragment via fragmentWriter.deleteFragment(id)
          - Emit 'reverie:reset:fragments' via switchboard
          - Return ok({ human: `Reset ${count} fragments. Self Model preserved.`, json: { reset: 'fragments', count }, raw: String(count) })

       b. handleResetSelfModel(args, flags):
          - Check _requireConfirm(flags). If err, return it immediately.
          - Call selfModel.coldStart() to reinitialize from seed prompt
          - Emit 'reverie:reset:self-model' via switchboard
          - Return ok({ human: 'Self Model reinitialized from cold start.', json: { reset: 'self-model' }, raw: 'self-model' })

       c. handleResetAll(args, flags):
          - Check _requireConfirm(flags). If err, return it immediately.
          - Reset fragments (same as handleResetFragments logic)
          - Reset Self Model (same as handleResetSelfModel logic)
          - Emit 'reverie:reset:all' via switchboard
          - Return ok({ human: `Factory reset complete. ${count} fragments deleted, Self Model reinitialized.`, json: { reset: 'all', fragments_deleted: count }, raw: 'all' })
