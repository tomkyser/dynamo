## Decisions Made

See key-decisions in frontmatter. The two most consequential decisions:

1. **Variable substitution is cosmetic debiasing (NO-GO).** The strongest finding from the research: transformers activate semantic neighborhoods from content, not labels. Replacing "Dual-Process Theory" with "$THEORY_A" while describing its characteristics still activates the Kahneman region of embedding space. No "context clearing" mechanism exists in transformer architecture. Replacement: adversarial counter-prompting that works WITH the model's attention mechanism.

2. **Inner Voice must be pure CJS with direct API calls (NO-GO on subagent).** The `type: "agent"` hook spawns a limited verification subagent (60s timeout, yes/no output only) that cannot produce injection text. The `claude -p` workaround has 5-15 second cold start exceeding all timing budgets except Stop. The correct pattern is already proven in Dynamo: `curation.cjs` and `sessions.cjs` make direct HTTP API calls from CJS command hooks.
