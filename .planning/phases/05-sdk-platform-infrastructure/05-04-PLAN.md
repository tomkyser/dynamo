          - Return `ok({ human: diagnosticsStr, json: { ...healthReport, impacted: analysis ? analysis.impacted : [] }, raw: JSON.stringify(healthReport, null, 2) })`.

          - If remoteResult.ok: `const newer = isNewerAvailable(local, remoteResult.value.version);`
          - Build result: `{ version: local, latest: remoteResult.ok ? remoteResult.value.version : 'unknown', updateAvailable: newer || false }`.
          - Return `ok({ human: \`Dynamo v${local}${newer ? ' (update available: v' + remoteResult.value.version + ')' : ''}\`, json: result, raw: local })`.

          - Steps: 1. Pull core via forge. 2. Update submodules via forge.submoduleUpdate(). 3. Verify health.
          - Return ok with update summary.
