    **CRITICAL: Association master table population (Research Pitfall 5)**
    Before writing fragments, collect ALL unique domain names, entity names, and attention tag names across all fragments in this formation batch. Queue upserts to master tables via Wire:

    For each unique domain name:
    ```javascript
    const domainEnvelope = createEnvelope({
      type: MESSAGE_TYPES.WRITE_INTENT,
      from: sessionContext.sessionId || 'reverie',
      to: 'ledger',
      payload: {
        table: 'domains',
        data: [{ id: domainName, name: domainName, created: new Date().toISOString(), fragment_count: 1 }],
        upsert: true, // Signal to Wire write coordinator: INSERT OR IGNORE
      },
      urgency: URGENCY_LEVELS.ACTIVE,
    });
    wire.queueWrite(domainEnvelope.value);
    ```
    Same pattern for `entities` table (id, name, type: 'auto', created) and `attention_tags` table (id, tag: tagName, created).
