<objective>
Deliver the Assay federated search service -- a query router and result merger that unifies search across all registered data providers (Ledger and Journal). Assay receives provider-specific queries, dispatches them in parallel via Promise.allSettled, tags every result with _provider metadata (name, type), and returns a flat result array with per-provider summary diagnostics.

Purpose: Assay is the unified search entry point that Reverie's recall pipeline (FRG-04) and sublimation scanning (SES-03) will consume. It enables consumers to search across heterogeneous data sources without knowing provider internals, while still supporting provider-specific optimization (SQL to Ledger, frontmatter scans to Journal).

Output: core/services/assay/assay.cjs (service factory) + core/services/assay/__tests__/assay.test.js (full test suite)
</objective>
