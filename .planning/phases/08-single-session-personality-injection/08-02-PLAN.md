      - `_currentFacePrompt = null` (in-memory cache for fast getInjection reads)
      - `_facePromptPath = path.join(resolvedDataDir, 'face-prompt.md')` where resolvedDataDir replaces `~` with `process.env.HOME`. Use `require('node:path')`.
      - `_checkpointDir = path.join(resolvedDataDir, 'data', 'checkpoints')`
      - `_initialized = false`

    **Methods:**

    `async init()`:
    1. Check if face prompt file exists at `_facePromptPath` via `lathe.readFile(_facePromptPath)`.
    2. If file exists (warm-start): read content, cache in `_currentFacePrompt`, return `ok({ source: 'warm-start' })`.
    3. If file does not exist (cold-start):
       a. Call `createColdStartSeed({ entropy })` to get sparse defaults.
       b. Save each aspect via `selfModel.save('identity-core', seed.identityCore)`, `selfModel.save('relational-model', seed.relationalModel)`, `selfModel.save('conditioning', seed.conditioning)`.
       c. Call `compose()` to generate and write the initial face prompt.
       d. Return `ok({ source: 'cold-start' })`.
    4. Set `_initialized = true`.

    `async compose()`:
    1. Get current budget phase from `_budgetTracker.getPhase()`.
    2. Call `_templateComposer.compose(phase)` to generate the face prompt text.
    3. Cache in `_currentFacePrompt`.
    4. Write to `_facePromptPath` via `lathe.writeFile(_facePromptPath, _currentFacePrompt)`.
    5. Emit `switchboard.emit('reverie:face-prompt-composed', { phase, tokens: Math.ceil(_currentFacePrompt.length / 4) })`.
    6. Return `ok({ phase, path: _facePromptPath })`.

    `getInjection()`: Return `_currentFacePrompt` directly (synchronous, no I/O). This is the hot path used on every UserPromptSubmit -- must be <1ms.

    `trackBytes(byteCount, source)`:
    1. Call `_budgetTracker.trackBytes(byteCount, source)`.
    2. If transition occurred (`result.changed === true`): call `compose()` (async -- fire and forget or await depending on context). Emit `switchboard.emit('reverie:budget-phase-changed', { from: result.from, to: result.to })`.
    3. Return the transition result.

    `getBudgetPhase()`: Return `_budgetTracker.getPhase()`.

    `getMicroNudge()`: If `_budgetTracker.getPhase() === 3`, return `_templateComposer.getMicroNudge()`. Otherwise return `null`.

    `async checkpoint()`:
    1. Build checkpoint object: `{ facePromptText: _currentFacePrompt, budgetPhase: _budgetTracker.getPhase(), cumulativeBytes: _budgetTracker.getStats().cumulativeBytes, attentionDirectives: selfModel.getAspect('conditioning')?.frontmatter?.attention_biases || {}, entropyState: entropy ? entropy.getState() : null, timestamp: new Date().toISOString() }`.
    2. Write to `path.join(_checkpointDir, 'compact-' + Date.now() + '.json')` via `lathe.writeFile(path, JSON.stringify(checkpoint, null, 2))`.
    3. Return `ok(checkpoint)`.

    `async resetAfterCompaction()`:
    1. Call `_budgetTracker.reset()` (uses DEFAULT_POST_COMPACTION_TOKENS * BYTES_PER_TOKEN).
    2. Call `compose()` to generate fresh face prompt for Phase 1.
    3. Emit `switchboard.emit('reverie:post-compaction-reset', { phase: _budgetTracker.getPhase() })`.
    4. Return `ok({ phase: _budgetTracker.getPhase() })`.

    `getSessionSnapshot()`: Return `{ budgetPhase: _budgetTracker.getPhase(), ...stats, entropyState: entropy ? entropy.getState() : null, facePromptPath: _facePromptPath }` where stats = `_budgetTracker.getStats()`.

    `async persistWarmStart()`:
    1. Write `_currentFacePrompt` to `_facePromptPath` via `lathe.writeFile()`. (Same file as compose writes -- dual purpose per D-02.)
    2. Return `ok({ path: _facePromptPath })`.

    Use `'use strict'` + CJS. Export: `{ CONTEXT_MANAGER_SHAPE, createContextManager }`.
