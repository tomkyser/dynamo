    - `createPulley(options = {})` function:
      - `const _commands = new Map()` -- stores `{ handler, description, flags, output }` per command name
      - `const _mcpTools = new Map()` -- stores `{ handler, schema, description }` per tool name

      - `registerCommand(name, handler, meta = {})`:
        - If `_commands.has(name)`: return `err('COMMAND_EXISTS', \`Command "${name}" already registered\`)`.
        - `_commands.set(name, { handler, description: meta.description || '', flags: meta.flags || {}, output: meta.output || ['human', 'json', 'raw'] })`.
        - Return `ok(undefined)`.

      - `route(positionals, argv)`:
        - Determine output mode: parse argv with `parseArgs({ args: argv, options: { json: { type: 'boolean', short: 'j', default: false }, raw: { type: 'boolean', default: false }, help: { type: 'boolean', short: 'h', default: false } }, allowPositionals: true, strict: false })`.
        - Extract outputMode: `values.json ? 'json' : values.raw ? 'raw' : 'human'`.
        - If `values.help` and positionals.length === 0: return `ok({ human: generateHelp(_commands), json: { commands: Object.fromEntries([..._commands].map(([k, v]) => [k, v.description])) }, raw: generateHelp(_commands) })`.
        - Find command: try longest match first. For i from positionals.length down to 1, check `_commands.has(positionals.slice(0, i).join(' '))`. Take first match.
        - If no match: return `err('COMMAND_NOT_FOUND', \`Unknown command: ${positionals.join(' ')}\`)`.
        - If `values.help`: return `ok({ human: generateCommandHelp(matchedName, _commands.get(matchedName)), json: { command: matchedName, ...meta }, raw: generateCommandHelp(matchedName, _commands.get(matchedName)) })`.
        - Extract remaining positionals after command name (for command-specific args).
        - Call `handler(remainingPositionals, { json: values.json, raw: values.raw })`.
        - If handler returns Err, return it.
        - Format: `const formatted = formatOutput(handlerResult.ok ? handlerResult.value : handlerResult, outputMode)`.
        - Return `ok({ formatted, outputMode })`.

      - `getCommands()`: return `ok(new Map(_commands))`.

      - `registerMcpTool(name, handler, schema)`:
        - If `_mcpTools.has(name)`: return `err('TOOL_EXISTS', \`MCP tool "${name}" already registered\`)`.
        - `_mcpTools.set(name, { handler, schema: schema || {}, description: schema && schema.description || '' })`.
        - Return `ok(undefined)`.

      - `getMcpTools()`: return `ok(new Map(_mcpTools))`.

      - Return via `createContract('pulley', PULLEY_SHAPE, { registerCommand, route, getCommands, registerMcpTool, getMcpTools, generateHelp: () => generateHelp(_commands) })`.

    - `module.exports = { createPulley, PULLEY_SHAPE }`.

    - `async function main(argv, pulley)`:
      - Parse global flags: `const { values: globalFlags, positionals } = parseArgs({ args: argv, options: { json: { type: 'boolean', short: 'j', default: false }, raw: { type: 'boolean', default: false }, help: { type: 'boolean', short: 'h', default: false } }, allowPositionals: true, strict: false })`.
      - Call `const result = await pulley.route(positionals, argv)`.
      - If `!result.ok`: `process.stderr.write(\`Error: ${result.error.message}\n\`); process.exitCode = 1; return result`.
      - Determine outputMode: `globalFlags.json ? 'json' : globalFlags.raw ? 'raw' : 'human'`.
      - If result.value.formatted: `process.stdout.write(result.value.formatted + '\n')`.
      - Return result.

    - `module.exports = { main }`.
