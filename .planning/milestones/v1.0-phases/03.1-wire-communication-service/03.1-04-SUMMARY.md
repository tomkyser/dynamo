## Decisions Made
- Wire factory composes internal modules rather than re-implementing -- delegates to registry, transport router, queue, and write coordinator
- MCP SDK CJS resolution uses absolute path fallback for StdioServerTransport and type schemas because Bun's require() does not match the SDK's wildcard package.json exports pattern
- Channel server uses setInterval-based polling (simpler lifecycle than recursive poll loop)
- Integration test uses _createServer({ port: 0 }) for random port assignment, consistent with relay-server.test.js pattern
