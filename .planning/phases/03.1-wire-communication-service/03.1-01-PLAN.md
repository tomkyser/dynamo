  <behavior>
    - createPriorityQueue({ backgroundQueueDepth: 5, activeQueueDepth: 10, directiveQueueDepth: 10 }) returns queue object
    - enqueue(envelope) adds envelope to appropriate urgency queue
    - dequeue() returns urgent envelope first, then directive, then active, then background
    - dequeue() on empty queue returns null
    - With 3 envelopes (background, directive, urgent) enqueued in that order, dequeue order is: urgent, directive, background
    - Within same urgency level, FIFO order is preserved
    - Background queue with depth limit 5: after 6 enqueues, oldest is dropped, queue.length is 5
    - Urgent queue has no depth limit (Infinity)
    - Directive queue depth configurable, defaults to 100
    - Active queue depth configurable, defaults to 200
    - Background queue depth configurable, defaults to 50
    - getDepth() returns object with count per urgency level
    - getDepth().total returns sum of all queues
    - isEmpty() returns true when all queues empty, false otherwise
    - flush() empties all queues and returns array of all flushed envelopes in priority order
  </behavior>
