Create core/armature/template-contracts.cjs:

1. `TEMPLATE_SLOT_TYPES` = Object.freeze(['string', 'array', 'number', 'boolean']).

2. `validateTemplateFrontmatter(frontmatter)`:
   - Returns `{ valid: boolean, errors: string[] }`.
   - Checks: `name` is non-empty string, `version` is non-empty string, `slots` is object.
   - For each slot: `required` must be boolean, `type` (if present) must be in TEMPLATE_SLOT_TYPES.
   - Returns `{ valid: true, errors: [] }` or `{ valid: false, errors: ['name: must be non-empty string', ...] }`.

3. `TEMPLATE_CONTRACT_SHAPE` = Object.freeze({ required: ['name', 'version', 'slots'], optional: ['description', 'tags', 'token_estimate', 'includes'] }).

Export: `module.exports = { validateTemplateFrontmatter, TEMPLATE_SLOT_TYPES, TEMPLATE_CONTRACT_SHAPE }`.

Create core/sdk/circuit/template-registry.cjs:

1. `createTemplateRegistry()`:
   - Internal state: `_templates = new Map()` (namespaced name -> Matrix).
   - Internal: `_linotype = require('../../../lib/linotype/linotype.cjs')`.
   - Internal: `_contracts = require('../../armature/template-contracts.cjs')`.

2. `async registerTemplates(manifest, moduleRoot)`:
   - `manifest` has `templates: { directory: string, namespace: string }`.
   - If no `templates` field, return early.
   - Build directory path: `path.join(moduleRoot, manifest.templates.directory)`.
   - Scan directory for `.md` files via `fs.readdirSync`.
   - For each `.md` file:
     a. Parse via `_linotype.parse(filePath)`.
     b. Validate frontmatter via `_contracts.validateTemplateFrontmatter(matrix.raw_frontmatter)`.
     c. If invalid: throw Error with file path and errors.
     d. Run `_linotype.validate(matrix, { registry: _templates })` for structural checks.
     e. If issues found: throw Error listing issues.
     f. Register as `${manifest.templates.namespace}:${matrix.name}`.
   - After all registered: re-validate includes (all `{{> partial}}` references resolve).
   - Log registered count.

3. `getTemplate(namespacedName)`:
   - Return `_templates.get(namespacedName)` or null.

4. `hasTemplate(namespacedName)`:
   - Return `_templates.has(namespacedName)`.

5. `castTemplate(namespacedName, context, options)`:
   - Get matrix from registry. If not found, throw Error `Template "${namespacedName}" not found in registry`.
   - Build partials Map from _templates for include resolution.
   - Call `_linotype.cast(matrix, context, { partials })`.
   - Return the Slug.

6. `listTemplates(namespace)`:
   - If namespace: filter _templates keys by prefix `${namespace}:`.
   - Else: return all keys.
   - Return as sorted array.

7. `clear()`:
   - _templates.clear(). For module disable cleanup.

Export: `module.exports = { createTemplateRegistry }`.

   getTemplate: (name) => templateRegistry.getTemplate(name),
   castTemplate: (name, context, options) => templateRegistry.castTemplate(name, context, options),

3. Add module lifecycle methods to Circuit:
   ```javascript
   async enableModule(moduleName) {
     // Resolve module from container's discovered modules
     const armature = container.resolve('armature');
     const moduleManifest = armature?.getModuleManifest?.(moduleName);
     if (!moduleManifest) throw new Error(`Module "${moduleName}" not found`);

     // Load module via module-discovery loadModule
     const moduleInstance = await loadModule(moduleManifest, { container, circuit: circuitApi, armature });

     // Register templates if manifest has templates section
     if (moduleManifest.templates) {
       await templateRegistry.registerTemplates(moduleManifest, moduleManifest._moduleRoot);
     }

     return { status: 'enabled', module: moduleName, instance: moduleInstance };
   },

   async disableModule(moduleName) {
     // Clear templates for this namespace
     // Call module cleanup if exists
     templateRegistry.clear(); // For v1 single-module, clear all is fine
     return { status: 'disabled', module: moduleName };
   }
   ```

4. Add 'registerTemplates', 'getTemplate', 'castTemplate', 'listTemplates', 'enableModule', 'disableModule' to the Circuit shape optional array.
