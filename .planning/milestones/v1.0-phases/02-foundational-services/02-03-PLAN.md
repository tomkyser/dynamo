    Factory: `createJsonProvider(options)` where `options = { lathe, filePath }`
    - `filePath` defaults to `.dynamo/state.json` relative to project root

    - `save(state)`: uses `lathe.writeFileAtomic(filePath, JSON.stringify(state, null, 2))` for crash safety. Before writing, copies current file to `filePath + '.bak'` via `lathe.writeFile(filePath + '.bak', currentContent)`. Debounced: does NOT write on every call. Instead, marks dirty and uses `setTimeout` to flush after 1000ms of inactivity. Always flushes immediately when called with `{ flush: true }` option (used by Magnet's stop()).
    - `clear(scope)`: removes scope key from state, saves
    - Self-validates via `validateProvider('json-provider', impl)`
    - Export: `module.exports = { createJsonProvider };`

    **2. `core/services/magnet/magnet.cjs`** -- State management service

    ```javascript
    'use strict';
    const { ok, err, createContract } = require('../../../lib/index.cjs');
    ```

    MAGNET_SHAPE: `required: ['init', 'start', 'stop', 'healthCheck', 'get', 'set', 'delete', 'getScope', 'registerProvider']`, optional: `['clearScope']`

    Factory: `createMagnet()`
    - Closure state:
      - `let _started = false;`
      - `let _switchboard = null;`
      - `let _provider = null;`
      - `let _state = { global: {}, session: {}, module: {} };`
    - `init(options)`: expects `{ switchboard, provider }`. Stores refs. Calls `await provider.load()` to hydrate _state. Returns Ok or Err.
    - `get(scope, ...args)`:
      - If scope === 'global': `get('global', key)` -> `_state.global[key]`
      - If scope === 'session': `get('session', sessionId, key)` -> `_state.session[sessionId]?.[key]`
      - If scope === 'module': `get('module', moduleName, key)` -> `_state.module[moduleName]?.[key]`
      - Returns the value directly (not Result-wrapped -- get is a read, not fallible)
    - `set(scope, ...args)`:
      - Captures oldValue via structuredClone before mutation
      - Global: `set('global', key, value)` -> `_state.global[key] = value`
      - Session: `set('session', sessionId, key, value)` -> `_state.session[sessionId] = _state.session[sessionId] || {}; _state.session[sessionId][key] = value`
      - Module: same pattern with `_state.module[moduleName]`
      - Emits to switchboard: `_switchboard.emit('state:changed', { scope, key: fullKey, oldValue, newValue: value })`
        - fullKey for global: just `key`
        - fullKey for session/module: `sessionId + '.' + key` or `moduleName + '.' + key`
      - Triggers provider save (debounced by provider)
      - Returns ok(undefined)
    - `delete(scope, ...args)`: similar to set but removes key, emits with `newValue: undefined`
    - `getScope(scope, namespace?)`:
      - Global: returns `{ ..._state.global }` (shallow copy)
      - Session: returns `{ ..._state.session[namespace] }` or `{}` if namespace missing
      - Module: returns `{ ..._state.module[namespace] }` or `{}` if namespace missing
    - `registerProvider(provider)`: validates via `validateProvider`, swaps _provider reference
    - `stop()`: calls `await _provider.save(_state, { flush: true })` to force immediate flush, sets _started = false
    - `healthCheck()`: returns `ok({ healthy: _started, name: 'magnet' })`
    - Self-validate: `return createContract('magnet', MAGNET_SHAPE, impl);`
    - Export: `module.exports = { createMagnet };`

    Per D-10: three-tier scoping (global/session/module)
    Per D-11: all mutations emit 'state:changed'
    Per D-12/D-13/D-15: lifecycle, options DI, self-validation
